# UrHome
Web application to improve the experience of finding rental houses


## 产品原型核心功能实现：
- 根据交通工具和时间得到地图上地理位置范围
- 标记处地图上的公共设施位置：健身房，超市，咖啡厅，餐馆
- 可以用颜色标出犯罪率和房价的分布


## 需要解决的问题：
- 如何区分neighborhood
- 如何在地图上添加颜色
- 如何获取犯罪率和房价的数据
- 设置交通工具和时间，得到地图上的范围
- 如何在地图上显示小图标
